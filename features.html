5 Tools. One Platform.

Every tool you need to hedge prediction markets.

From historical replay to auto-optimized multi-leg constructions. Each calculator builds on the previous one.

01 — Position Backtester

Replay prediction markets with real price history.

Track historical YES/NO pricing, overlay crypto spot prices (BTC/ETH/SOL), and study how each market evolved from open to resolution.

  • Historical YES/NO price charting for any Polymarket event
  • Crypto price overlay (BTC, ETH, SOL, XRP via Binance)
  • Sum display toggle for multi-outcome markets
  • Visual correlation between underlying and prediction market pricing
Try Backtester

02 — Position Builder

Project P&L with Black-Scholes pricing.

Paste any Polymarket event URL. See projected payoff curves using European Binary and One-Touch Barrier option models. Stress-test with IV multiplier.

  • Black-Scholes European Binary option pricing
  • One-Touch Barrier pricing for "hit" events
  • IV Multiplier slider (x0.25 to x4.00) for stress testing
  • Construction cost projection and JPG snapshot export
Try Builder
P&L Projection — Black-Scholes IV x1.5
03 — Position Hedger

Build multi-leg portfolios combining PM + Bybit.

The core tool. Add Polymarket positions and Bybit options into a single portfolio. See combined P&L across 4 time snapshots. Full Bybit option chain with bid/ask/delta.

  • Multi-leg portfolio: PM binary + Bybit vanilla options
  • Full Bybit BTC option chain (Bid/Ask/Mid/Delta)
  • 4 time curves: Now, +7d, +14d, At Expiry
  • JSON snapshot save/load + PNG chart export
  • "Refresh prices" for live re-calculation
Launch Hedger
Combined P&L — 3 Legs Live
04 — Position Designer

Auto-find the best hedge for each PM position.

Paste your PM NO position. The optimizer scans every available Bybit option and finds the single best hedge for 3 volatility scenarios.

  • 3 optimization columns: +/-1%, +/-10%, +/-20% ranges
  • Average expected profit for each volatility scenario
  • Results table with best hedge per strike
  • "Open in Position Hedger" cross-link for detailed inspection

05 — Position Auto-Hedger

Optimal 2-leg spread hedges. Automatically.

The most advanced tool. Creates PM strike pairs, finds different calls/puts on Bybit for each, and calculates max drawdown, avg profit, and avg loss for the full BTC price plane.

  • 3-leg structure: PM NO + Long Option + Short Option
  • PM strike pair optimization (80-60, 85-55, etc.)
  • Full BTC price plane analysis: max drawdown, avg profit, avg loss
  • 6-leg double-sided constructions for bidirectional profit
  • "Open in Position Hedger" for manual tweaking
